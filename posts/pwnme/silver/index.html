[PwnMe 2600] Forensic - Silver

May 6, 2023 · 2 min ·  
Hey,  
I received a flash drive in my mailbox.
It appears that the person who sent me the package was aware that I was using Linux.
However, the flash drive self-destructed before I was able to make an image of it.
To get the flag, you have to go to the showcase site of the c2 in question.  
  
This is real malware, so be careful when running it, use a vm for malware analysis

Let’s import this image in Autopsy (Forensic Software), it find 3 interesting files : * 1 script .sh * 1 Pdf * 1 executable We can see that the script install an executable “firefox.elf” in the firefox path of the victim computer and then delete the PDF, who must be the entry point of the malware.

We extract “firefox.elf” and upload it on VirusTotal to learn more about it. That definitely our malware, now how can we find out it behaviour ? Two options, the first one do reverse engineering and the second one easiest and funnier launch it !

To do so, let’s launch it on a virtual machine, while recording all the network traffic using Wireshark. Doing so, we can see two things, a destination IP 178.62.67.181 and by following the HTTP Flow we notice a connection on 443 port. We connect to the IP using 443 port. And we can see an image ! We got the Flag : PWNME{1_L0V3_5L1V3R}
