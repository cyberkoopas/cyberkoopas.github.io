Solution#

Firstly, we can convert the ASCII message file to a binary file (message.bin).
It’s possible with CyberChef. Select operation From Binary and download file.
Cyberchef parameters :

  • Delimiter: None
  • Byte Length : 8

We know the flag format is dvCTF{REDACTED}, we can perform a XOR Known plaintext attack.
The tool to get the XOR key and decrypt file is : xorknown.py.
Known plaintext : dvCTF{.

python2 xorknown.py message.bin "dvCTF{"
# Key length: 6 
# Partial Key: CR9PT0 
# Plaintext: You did it! The flag is dvCTF{80R3D_K3Y_15_K3Y_MY_FR13ND}. Well done !
